### Installation:
Node.js version 16 or higher is required.

Clone the project, then go to the main(root) directory in terminal and use commands below:

## Setup

```bash
make setup
```

## Run
Open terminals and run separately FE & BE:
```bash
$ make start-frontend
$ make start-backend
```
